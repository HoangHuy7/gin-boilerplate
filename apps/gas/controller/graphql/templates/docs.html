📚 Các Module API

🛍️ Sản phẩm (Product)

Quản lý thuốc trừ sâu, phân bón, thức ăn chăn nuôi

  • products - Danh sách sản phẩm
  • createProduct - Thêm sản phẩm
  • updateProduct - Cập nhật
  • deleteProduct - Xóa

👥 Khách hàng (Customer)

Quản lý khách hàng và công nợ

  • customers - Danh sách khách hàng
  • createCustomer - Thêm khách hàng
  • total_debt - Theo dõi công nợ

🛒 Đơn hàng (Order)

Bán hàng, tính tiền, ghi nợ

  • orders - Danh sách đơn hàng
  • createOrder - Tạo đơn hàng
  • total_amount - Tổng tiền
  • debt_amount - Tiền nợ

📦 Kho (Inventory)

Nhập xuất kho, theo dõi tồn kho

  • inventoryLogs - Lịch sử nhập xuất
  • createInventoryLog - Ghi nhận

💰 Công nợ (Debt)

Giao dịch nợ và thanh toán

  • debtTransactions - Lịch sử giao dịch
  • createDebtTransaction - Ghi nhận

🚚 Giao hàng (Delivery)

Quản lý giao hàng trong ngày

  • deliveries - Danh sách giao hàng
  • todayDeliveries - Giao hàng hôm nay

📝 Ví dụ Query

# Lấy danh sách sản phẩm
query {
  products {
    id
    name
    price
    stock_quantity
    barcode
  }
}
# Tạo đơn hàng mới
mutation {
  createOrder(input: {
    customerId: "uuid-here"
    items: [
      { productId: "uuid", quantity: 2, price: 35000 }
    ]
    paidAmount: 70000
  }) {
    id
    totalAmount
    status
  }
}

🔧 Scalar Types

  • ID - UUID identifier
  • String - Chuỗi văn bản
  • Int - Số nguyên
  • Decimal - Số thập phân (tiền tệ)
  • Time - Thời gian ISO8601
  • Boolean - true/false
